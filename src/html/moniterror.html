Monitoring Service Not Running;

Why am I seeing this message?

SoftNAS® has detected that monitoring service is not running when trying to contact the server system over the network. This can happen when there is a syntax error in config file /etc/monit.conf, that makes monitoring service unable to start.

What can I do to troubleshoot this issue?

You can try to restart monitoring service. Open System services -> Select Monit service -> Click restart -> Reload this frame

If the problem persists, open administrator settings -> Click monitoring tab -> Correct monitoring settings -> Save (if everything is ok you will see success message) -> Reload this frame.

If the problem persists, you can Contact Support if you are unsure why this is happening or need assistance to resolve it.

Click Here to reload this frame after the issue has been resolved or to try again.
